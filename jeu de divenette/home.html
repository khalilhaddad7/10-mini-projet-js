Guess the Number

Guess a number between 1 and 100

Un nombre aléatoire entre 1 et 100 est généré lorsque la page est chargée. L'utilisateur entre une devinette dans un champ (par exemple, un nombre comme "42"). Lorsqu'il clique sur un bouton pour soumettre sa réponse, la fonction checkGuess est exécutée : Si la devinette est exacte, un message de félicitations s'affiche. Si la devinette est trop basse, un message indique "Too low! Try again." Si la devinette est trop haute, un message indique "Too high! Try again."
